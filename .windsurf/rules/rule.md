# 数据库
- 项目使用的是 PGSQL，必须通过 MCP pgsql 工具操作，禁止生成 SQL 文件
- PGSQL 连接配置统一从环境变量读取，禁止硬编码连接信息
- 本地 PGSQL 连接信息：host=127.0.0.1 port=5432 user=root password=root sslmode=disable，PostgreSQL 18.3（Homebrew），位于 /opt/homebrew/Cellar/postgresql@18/18.3 pg的id自增 最好定义成: id bigint GENERATED BY DEFAULT AS IDENTITY NOT NULL
- 项目使用Gin-Vue-Admin
- 本项目默认使用 pg-local MCP 工具換作数据库，不要生成 SQL 文件
- apifox 使用mcp apifox-test
