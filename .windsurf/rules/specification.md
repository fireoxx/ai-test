mysql数据库建表规范：
1.所有表必须加not null default，text、json类型除外
2.必须加上created_at、updated_at，按下面的例子创建
例子：
`created_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间'
`updated_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
3.表字段需要加上comment
4.datetime类型字段默认值不能设置current_timestamp的，默认值：1970-01-01 08:00:00
5.date类型字段默认值：1970-01-01
6.double、float类型的字段改成decimal类型
7.不允有外键约束，外键约束必须由业务代码实现



pgsql数据库表建表规范：
1.所有表必须加not null default，text、jsonb类型除外
2.必须加上created_at、updated_at，按下面的例子创建
例子：
created_at timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间'
updated_at timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '更新时间'
3.表字段需要加上comment
4.timestamp类型字段默认值不能设置current_timestamp的，默认值：1970-01-01 08:00:00
5.id主键：id bigint GENERATED BY DEFAULT AS IDENTITY NOT NULL
6.小数类型统一使用decimal类型，不要使用float、double类型
7.不允有外键约束，外键约束必须由业务代码实现
8.字符串统一使用varchar类型
9.时间类型统一使用timestamp类型
10.存储YYYY-MM-DD，统一使用date类型
11.禁止创建触发器  禁止创建触发器  禁止创建触发器
